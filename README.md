# File Service

File and media handling service.

## Overview
Manages file uploads and downloads with validation and streaming support.

## Features
- Streaming file uploads
- File type and size validation
- S3-compatible object storage
- Signed URL support

## Tech Stack
- Backend: Go
- Storage: S3-compatible storage

## Architecture
- Streaming-based IO handling
- Storage abstraction layer

## Notes
Optimized for performance and memory safety.
